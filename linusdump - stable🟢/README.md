# FreezeHost Python Discord Bot Setup

1. Upload all your files to FreezeHost.
2. Set your Discord bot token as an environment variable named DISCORD_TOKEN.
3. Install requirements:
   pip install -r requirements.txt
4. Make sure 'lune' is installed and available in PATH (or adjust the subprocess command in main.py).
5. Start the bot:
   python main.py

The .linusdump command will run your dumper and reply with the output or an error message.
